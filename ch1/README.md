# 리액트 프로젝트 시작하기

## 리액트란 무엇인가

- 자동으로 업데이트 되는 UI
- UI = render(state)
- render 함수는 순수 함수로 작성
  - 랜덤 함수 사용 x
  - 외부 상태 변경 x
  - 입력이 같으면 출력이 같다는 말.
- state는 불변 변수로 관리
- 가상 돔(virtual dom)
